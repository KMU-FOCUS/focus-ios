🎥 FOCUS

Privacy Protection Live Streaming Platform for Mobile Streamers

FOCUS는 라이브 스트리밍 환경에서 제3자의 얼굴 프라이버시를 보호하기 위해 개발된 iOS 애플리케이션입니다.

방송 중 카메라 화면에 포착되는 제3자를 실시간으로 감지하고, 얼굴 분석 결과를 기반으로 아바타 치환 및 프라이버시 보호 처리를 수행합니다.

이를 통해 스트리머와 주변 시민 모두가 보다 안전한 환경에서 라이브 스트리밍을 진행할 수 있도록 지원합니다.
